# Awesome Meccha Chameleon Hide Spots

> A community-curated awesome list of hiding spots for [Meccha Chameleon](https://store.steampowered.com/app/4704690/) — the paint-based hide-and-seek game by LEMORION.

🌐 **Live site**: https://mecchachameleon.art/ — fan-made browser companion with 50+ spot breakdowns.

---

## What is this?

This is a community-driven awesome list. Hiders, add your best finds. Seekers, contribute counter-tips. Together we make a complete atlas of every hiding spot across all official and workshop maps.

## Why contribute?

- 🎯 Get featured on the [Meccha Chameleon Art](https://mecchachameleon.art/) site (dofollow backlink)
- 🏆 Build your profile in the Meccha Chameleon community
- 📚 Help new players learn camo strategy faster
- 🤝 Network with other players

## How to contribute

1. **Fork** this repo
2. **Add your spot** to the relevant map section below (or create a new map section)
3. **Submit a Pull Request** with:
   - Map name
   - Spot location (e.g. "vintage room, behind sofa, NE corner")
   - Screenshot (PNG/JPG, < 2MB)
   - Difficulty: 🟢 easy / 🟡 medium / 🔴 hard
   - Best paint colors (RGB or hex)
   - Seeker counter-tip
4. **Wait for review** — we merge within 48h

Use this template for each entry:
```markdown
### [Map] - [Location]
- Difficulty: 🟢/🟡/🔴
- Best paint: #hex1, #hex2, #hex3
- Screenshot: ![spot](./spots/your-spot.png)
- Counter-tip: ...
- Submitter: @github-username
```
